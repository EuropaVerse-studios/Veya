# Veya

**Veya** is a modern, lightweight, and versatile programming language designed for simplicity, readability, and power.  
Inspired by languages like C, Python, and Haxe, Veya aims to provide a flexible platform for developers to create software efficiently.

---

## Features
- **Readable syntax** – Easy to learn and write.  
- **Dynamic typing** – No need for verbose type declarations.  
- **Extensible** – Easy to add new features and modules.  
- **Lightweight interpreter** – Fast startup and minimal resource usage.  
- **Open Source** – Free to use and modify with proper credit.
